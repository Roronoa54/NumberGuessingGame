# **NumberGuessingGame**
A console-based number guessing game built in C++ as a beginner project 
to practice core programming fundamentals and good coding habits.

## **Current Status**
This project is currently in development

## **Planned Features**
- Random number generation between 1-100
- "Higher" and "Lower" hints
- A counter that counts attempts made
- Difficulty modes which limit guesses the user can make (Easy, Medium, Hard)
- Play Again option
- Input Validation
- Leaderboard

## **What I'm practicing**
- Clean coding habits (meaningful names, constants and no magic numbers)
- Proper use of functions and not dumping everything into the main()
- Input Validation
- Get familiar with github basics

## **Installation**

### **Clone**
```bash
git clone https://github.com/Roronoa54/NumberGuessingGame.git
```

### **Manual**
1. Download ZIP file (Code->Download ZIP)
2. Extract the folder
3. Open the sln file in Visual Studio
4. Press Ctrl+F5 to run
