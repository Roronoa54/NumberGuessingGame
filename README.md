# **NumberGuessingGame**
A console-based number guessing game built in C++ as a beginner project 
to practice core programming fundamentals and good coding habits.

## **Current Status**
This project is currently in development. So far, A random number generator has been created along with a number guesser which takes in the user input.
The input gets validated and gives a hint to the user based on the number being high or low.
An attempt counter has also been added which counts how many attempts it took user before they finally guessed the correct number and the play again option.

### **To do list**

## **Installation**

### **Clone**
```bash
git clone https://github.com/Roronoa54/NumberGuessingGame.git
```

### **Manual**
1. Download ZIP file (Code->Download ZIP)
2. Extract the folder
3. Open the sln file in Visual Studio
4. Press Ctrl+F5 to run
